# ESP32CAM_RTSP
ESP32-CAMとRTSPプロトコルを利用した簡易ネットワークカメラ。
python等で映像を受け取り利用することができる。

## 使い方
1. esp32-cam/src/wifi_config.hに接続するWiFiの情報を記述。
1. esp32_rtsp.inoをESP32-CAMに書き込む。
